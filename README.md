# Market Pulse India V1

Zero-cost starter project designed for tablet deployment.

## Included
- Responsive premium landing page
- TOP NEWS / ALL NEWS UI
- Search
- Exact-original-headline display structure
- Original-source links
- ₹200 subscription UI (payment NOT configured)
- Cloudflare Worker starter for permitted news sources
- No fake news and no AI rewriting

## Important
This ZIP does not pretend to provide live news yet. Connect only sources whose terms permit your intended use. A commercial subscription product may require source licenses.

## Tablet deployment
1. Create a GitHub account/repository.
2. Upload all files from this project.
3. Connect the repository to Cloudflare Pages for the static site.
4. Deploy `worker.js` separately as a Cloudflare Worker.
5. Add permitted official feeds/APIs one at a time.
